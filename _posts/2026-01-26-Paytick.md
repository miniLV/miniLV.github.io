💡 **背景**: 有时候你会迷茫，上班的意义是什么，特别是工作多年之后。当你感觉班上不下去的时候，可能会想说自己上班的意义是什么。这时候，你需要知道自己每分钟的价值，才能有继续工作的动力。于是就有了这个工具 —— 一个实时显示你当前已赚多少钱的小玩意儿~

<br>

## 🤖 关于 Vibe Coding

这个项目有点特别，是一次 **100% Vibe Coding** 的实践。

整个开发过程中，我的角色是产品经理 —— 负责提需求、定方向、做决策。代码实现 100% 由 **Claude Code (Opus 4.5)** 完成，UI 设计则是用 Figma AI 自动生成后微调。

同时，这也是一次 **Spec-Driven Development (SDD)** 的实操。感兴趣可以看看 [github/spec-kit](https://github.com/github/spec-kit)。

整个过程下来效率确实高了不少，这种开发方式值得尝试。

## ✨ 项目功能

- [x] 实时收入显示 — 菜单栏直接看，精确到分
- [x] 愿望清单 — 想买啥设置啥，看进度条慢慢填满
- [x] 加班提醒 — 过了下班时间界面变红，提醒你时薪正在被稀释
- [x] 上下班通知 — 到点自动弹出系统通知，提醒你该上班/下班了
- [x] 隐私保护 — 同事凑过来看屏幕的时候一键切换，支持快捷键
  - Emoji模式：数字变成 🍎🚀⭐
  - 模糊模式：高斯模糊
  - 打码模式：显示 ****
- [x] 多语言支持 — 在 Settings → Advanced 中切换语言，支持中文/英文

![Language Settings](/img/in-post/paytick-language-settings.png)



## 🛠️ 技术栈

- Swift 6.0
- SwiftUI（纯代码）
- macOS菜单栏应用
- 本地数据存储
- 多语言本地化（中文/英文）
- Apple公证签名
- Figma AI 生成 UI



## 📦 使用方法

1. 去 [Releases](https://github.com/miniLV/Paytick/releases) 下载 `Paytick.dmg`
2. 拖进应用程序文件夹
3. 输入月薪、工作天数、上下班时间
4. 看数字跳

已通过Apple公证，双击就能跑。如果提示无法验证，右键选"打开"就行。
