## About Vibe Coding

This project is special ‚Äî it's my first **100% Vibe Coding** experiment.

What does that mean? From start to finish, the code was basically written by AI. My job was to provide requirements, review, and steer the direction. The UI was also auto-generated by Figma AI ‚Äî I just tweaked some details. The whole process was pretty smooth, and definitely more efficient than before.

Of course, you can't completely turn off your brain. You still need to review carefully, or AI might introduce some bugs. But overall, this development approach is worth trying.

## How to Use

1. Download `Paytick.dmg` from [Releases](https://github.com/miniLV/Paytick/releases)
2. Drag to Applications folder
3. Enter your salary, work days, and work hours
4. Watch the numbers tick

Apple notarized, double-click to run. If you see "unverified developer", right-click and select "Open".

<br>

## Privacy

All data stored locally. No network, no upload, no tracking. Code is open source ‚Äî check it yourself if you want.
